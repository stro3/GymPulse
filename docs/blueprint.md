# **App Name**: GymPulse

## Core Features:

- User Dashboard: Display user profile, membership details, workout/diet plans, class bookings, progress tracking, payments, and notifications.
- Trainer Dashboard: Display trainer profile, assigned members, workout plan creation tool, diet plans, class schedule management, attendance tracking, and messaging.
- Admin Dashboard: Enable user and trainer management, membership plan management, class scheduling, payment tracking, analytics, and content management.
- Role-Based Login: Implement role-based login (User, Trainer, Admin) with secure authentication (Google login, OTP, or email verification).
- Personalized Fitness Tool Suggestions: AI-powered tool to generate personalized workout and nutrition plans, and recommend appropriate gym equipment based on user profiles and fitness goals.
- Mobile Responsive Design: Ensure the dashboard works seamlessly on phones and tablets with a responsive layout.
- Interactive BMI Calculator: An integrated BMI calculator, within the user dashboard, to let users quickly assess their Body Mass Index (BMI) to inform and guide fitness goals.

## Style Guidelines:

- Primary color: Vibrant red (#E63946) to convey energy and motivation.
- Background color: Light gray (#F1FAEE), a desaturated shade of the primary, to provide a clean and modern backdrop.
- Accent color: Light blue (#A8DADC) to highlight interactive elements and CTAs.
- Body and headline font: 'Inter' sans-serif font for a clean, modern, and readable experience.
- Use modern, outlined icons related to fitness activities, user profiles, and dashboard functionalities.
- Maintain a clean and structured layout with clear section divisions for easy navigation across all dashboards.
- Implement subtle transitions and animations for loading indicators and feedback on user actions to enhance the user experience.
